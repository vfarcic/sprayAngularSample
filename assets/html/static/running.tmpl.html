Running the application

Prerequisites

Back-end

Following back-end prerequisites need to be installed:
  • Scala
  • JDK 7+
  • SBT (Simple Build Tool)
  • MongoDB

Front-end

Following front-end prerequisites need to be installed:
  • NodeJS with NPM
  • Grunt: npm install -g grunt-cli

Dependencies

Back-end dependencies will be download first time the application is run. To download the front-end dependencies: 
cd assets
npm install

Running

MongoDB needs to be up and running: 
mongod
To run the application: 
sbt run
Application can be run in "triggered restart" mode. Your application starts up and SBT watches for changes in your source (or resource) files. If a change is detected SBT recompiles the required classes and sbt-revolver automatically restarts your application. When you press <ENTER> SBT leaves "triggered restart" and returns to the normal prompt keeping your application running. 
sbt ~re-start
Once the application is running, open the http://localhost:8080.
