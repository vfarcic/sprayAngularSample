Development: running

To run the application in development mode we need to start the MongoDB and run the application using SBT.

MongoDB

We'll start MongoDB using default settings. 

mongod

SBT

The application is already configured to use SBT Revolver. It is a plugin for SBT enabling a super-fast development turnaround for Scala applications. Whenever any part of the code changes, SBT Revolver will recompile that change and we'll see the result almost instantly. There is no need for any additional task to be performed (packaging, deployment, etc). 

sbt ~re-start

With MongoDB and SBT Revolver up and running, we can see the application by opening http://localhost:8080/.
