Front-end: static screens

This application contains both static and dynamic screens. Static screens contain plain HTML, CSS and Javascript code that does not communicate with any server.

Following steps describe how to add a new static screen.

Create a link

Create a link to the new page with the href="/page/static/[NAME]". [NAME] can be any alpha-numeric string. Spaces are not allowed.

Following example is taken from the menu.tmpl.html: 

<li><a href="/page/static/newStaticScreen">Static Screen</a></li>

Template

Create a template file named [NAME].tmpl.html in the assets/static directory. For examples of how to write contents of the newly created file, please take a look at the existing templates in the assets/static directory.

Open

Open the newly created template by navigating to the http://localhost:8080/page/static/[NAME]. The screen you are looking at right now is an example of static HTML template.
