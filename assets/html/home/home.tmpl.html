Welcome to the Angular/Spray sample application

This application uses AngularJS, Bootstrap CSS, Spray, AKKA and MongoDB frameworks. Behind those frameworks are JavaScript, Scala and Java languages.

The intent behind this sample application is to facilitate easier startup into the world of reactive programming.

Features of the application are divided into following sections:

Running

First step is to run the application. Instructions can be found in the Running the aplication screen.

Presentation

Presentation layer is done with AngularJS and Bootstrap CSS. All communication with the server is done using REST API. This layer is decoupled from the server.
